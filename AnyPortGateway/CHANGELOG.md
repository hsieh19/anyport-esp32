# AnyPort ESP32 固件更新日志 (Changelog)

所有对 AnyPort ESP32 网关固件的重大更改都将记录在此文件中。
  
## [1.2.1] - 2026-03-19
### 新增
- **实时报文监听 (Sniffer)**：从站模拟器模式新增 Web 实时报文显示窗口，支持 HEX 报文流捕获、TX/RX 染色区分及 `分:秒.毫秒` 级高精度时间戳。支持“一键清空硬件缓冲区”与“前端暂停接收”功能。
- **大地址智能映射**：重构了 Modbus 地址解析引擎，支持 0-65535 全量地址自动识别。兼容“字面量绝对地址”（如 40600）与“标准偏移地址”（如 599）的自动对齐，解决了非标上位机的兼容性痛点。

### 优化
- **模拟器响应性能**：优化了在高波特率下的 Modbus 响应 Hook 逻辑，降低了监控开启时的系统延迟。
- **UI 交互**：优化了寄存器列表的刷新频率与状态显示。

## [1.2.0] - 2026-03-14
### 新增
- **USB 转 RS485 透传模式 (Mode 3)**：支持将 ESP32-C3 作为 USB-to-RS485 高速桥接器使用。针对 ESP32-C3 简约版优化，支持通过 Web 页面手动配置串口参数（波特率、数据位、校验位、停止位）。
- **mDNS 域名访问**：支持通过 `anyport.local` (默认) 或自定义域名在局域网内访问 Web 配置面板，无需记忆 IP 地址。
- **零配置优化**：默认透传参数设为 9600 8N1，符合大多数工业设备初始设置。

## [1.1.0] - 2026-03-13

### 新增
- **从站模拟器模式**：支持 0x/1x/3x/4x 寄存器模拟，包含动态值模拟逻辑（随机、递增、递减）。
- **复合 Web 面板**：支持在网关模式与模拟器模式间动态切换及配置寄存器。
- **版本可视化**：在 Web 控制面板的页面标题（Title）和主标题（H1）中增加了固件版本号显示。

### 优化
- 完善了代码审计中发现的模块化建议。
- 优化了 MQTT 和网络初始化的稳定性。

---

## [1.0.0] - 2026-03-10

### 初始发布
- **MQTT 网关模式**：实现 Modbus RTU (RS485) 与 Modbus TCP (W5500) 协议转发至 MQTT。
- **网络管理**：集成 WiFi STA/AP 切换，支持 W5500 以太网静态 IP 配置。
- **Web 配置**：内置基础 HTTP 管理面板，用于设置网络参数与 MQTT 账户。
- **时间同步**：支持 NTP 网络时间校验。
